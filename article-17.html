# 🌈 CSS display:flex 공부하기 - Items 속성 1탄 ### ✅ order : 아이템들의 나열순서 ### ✅ flex : flex-grow/shrink/basis 를 한번에 쓰는 축약형 ### ✅ flex-grow : 기본크기보다 커질 수 있는 결정 ### ✅ flex-shrink : 기본크기보다 작아질 수 있는 결정 ### ✅ flex-basis : flex아이템의 기본 크기 설정 ### ✅ ailgn-self : 해당 아이템의 수직축 방향 정렬

# 📐order ## 아이템에 숫자를 지정해 순서를 정합니다. #### 작은 숫자일 수록 먼저 배치됩니다. #### 👉 #1 {order : 3} #### 👉 #2 {order : 1} #### 👉 #3 {order : 2}
``` 2️⃣ 3️⃣ 1️⃣ ```
# 📐flex ## flex-grow/shrink/basis 를 한번에 쓰는 축약형입니다 #### 아이템의 너비(증가/감소/기본)을 설정합니다 #### 👉 flex : 1; ``` flex-grow : 1; flex-shrink: 1; flex-basis: 0%; ---> flex-basis의 기본값은 auto이지만, 값을 생략하면 0으로 적용됩니다! ```
#### 👉 flex: 1 1 auto; ``` flex-grow : 1; flex-shrink: 1; flex-basis: auto; ``` #### 👉 flex: 1 500px;
``` flex-grow : 1; flex-shrink: 1; flex-basis: 500px; ```

## `flex-grow` #### 아이템의 증가 너비 비율을 설정합니다. ##### 숫자가 클 수록 너비가 커지고, 값이 0일 경우는 효과가 없습니다.
#### 👉 아이템 3개의 너비가 2,1,1 이라면
``` 1️⃣1️⃣ 2️⃣ 3️⃣ 아이템 1 : 너비의 50% 아이템 2 : 너비의 25% 아이템 3 : 너비의 25% ```

## `flex-shrink` #### 아이템의 감소 너비 비율을 설정합니다. ##### 👉 0보다 큰 값이 셋팅 되면 해당 아이템이 유연하게 변하면서 기본크기(basis)보다 작아집니다 ##### 👉 0으로 셋팅하면 기본크기 밑으로 작아지지 않기 때문에 고정크기의 컬럼을 쉽게 만들 수 있어요!
``` .item{ flex-shrink : 0; width : 100px; -----> 고정크기는 width로 설정! } ```

## `flex-basis` #### 아이템의 기본크기를 설정합니다. (기본적으로 차지하는 크기) ##### 👉 row일때는 너비, cloumn일때는 높이! ##### 👉 width와 함께 사용하면 설정한 크기로 보여지지만, 기본크기는 basis로 설정되어있습니다!


# 📐align-self ## 교차축에서 개별 item 정렬방법을 설정합니다. #### 필요에 의해 일부 item만 정렬을 변경하려고 할 경우 align-self를 사용할 수 있습니다. #### align-items 적용보다 우선권이 있습니다.
#### 👉 auto (기본값) : align-items의 속성을 상속받습니다.
#### 👉 stretch : 교차축을 채우기 위해 item을 늘립니다.
#### 👉 flex-start : 시작점에 정렬
#### 👉 flex-end : 끝점에 정렬
#### 👉 center : 가운데 정렬
#### 👉 baseline : item을 문자 기준선에 정렬
