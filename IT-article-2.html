# 🚩내가 보려고 만드는 MySQL 기본명령어 모음집 -- 1탄 # 👀 데이터 조회하기! 설명순서는 큰 단위에서 작은 단위 순으로! ***

## 🔎 데이터베이스 조회 (a.k.a 데이터가 모여있는 폴더) ## `SHOW DATABASES;` ##### 👉 생성되어 있는 데이터베이스의 목록을 조회합니다.
## 🔎 테이블 조회 (a.k.a 하나의 엑셀파일) ## `SHOW TABLES`; ##### 👉 선택되어 있는 데이터베이스 안에 만들어져 있는 테이블들의 목록을 조회합니다.
``` ✅ USE a1; ---- 'a1' 이라는 데이터베이스를 선택합니다. ```
## `DESC '테이블명';` ##### 👉 선택한 테이블의 칼럼구조와 속성을 조회합니다.


## 🔎 ROW(로우) 조회를 위한 명령어 (a.k.a 엑셀파일의 한 행, 실질적인 데이터가 들어있는 곳) ## `SELECT id;` ##### 👉 id 칼럼의 row의 값을 선택하여 조회합니다.
``` ✅ SELECT id FROM article; ```
